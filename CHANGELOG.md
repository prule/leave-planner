# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [1.1.0] - 2026-03-01

### Added
- Implemented automatic reloading for the PWA when a new version is available.
- Added Work From Home day tracking
- Configurable Financial Year start month
- Added financial year totals
- Added public holidays datasource for showing on WFH selector
- Show beginning of year on chart
- Updated About page

### Changed
- Upgraded project dependencies.
- Updated code to be compatible with Tailwind 4.

### Fixed
- Resolved compilation errors after dependency upgrades.

## [1.0.0] - 2026-02-28

### Added
- Monthly spreadsheet view to track leave balance.
- Ability to manage leave entries with descriptions.
- A line graph to visualize projected leave balance over time.
- Settings page to configure initial balance, accrual rates, and projection horizon.
- Data export to JSON/CSV.
- PWA support for installation on devices.
